bloxels edu

a little history

Bloxels EDU is the third and latest incarnation of the brand that began with "Bloxels Builder" at Pixel Press in 2015. Like its predecessors, players build platformers from a few basic elements that correspond to 8 colored blocks placed on the physical Bloxels game board grid. Aside from the physical component, one of the more unique aspects of Bloxels is giving players simple tools for creating and sharing every piece of art used in their games.

my role

For this release, I rewrote all of the core platformer code, implemented and tested different approaches to client-side user data serialization, and did a lot of UX and game design iteration alongside my colleague Aaron Clark, the artist/animator/designer who was brought onto the team during the Star Wars project. All of my work has involved a substantial focus on optimization in general.

After all my time at Pixel Press, I'm most proud of my contributions to the unreleased version 2.0. This page's header video provides a little taste. That work included implementing a pixel art refresh of the entire app's UI, as well as building/designing a new and robustly featured game editor. For gameplay, I'd added variants for enemy AI and player attacks, as well as configurable options for the size, speed, jump, max health, and visual effects for any character.

private void ApplyMovement()
{
	float currentGravity = _gravityCalculation.Invoke();

	// Calculate position delta due to gravity using simplified Velocity Verlet, and update position accordingly:
	Vector2 gravityOffset = new Vector2(0f, 0.5f * currentGravity * unityTime.fixedDeltaTime * unityTime.fixedDeltaTime);
	Vector2 adjustedVelocity = controller.Move((velocity * unityTime.fixedDeltaTime) + gravityOffset, unityTime.fixedDeltaTime);

	//If we hit a ceiling during the move, zero out the vertical velocity:
	if (controller.isTouchingCeiling && velocity.y > 0f)
	{
		velocity.y = 0f;
	}
	else  // Otherwise, update vertical velocity based on current gravity:
	{
		velocity.y += currentGravity * unityTime.fixedDeltaTime;
		if (velocity.y <= movement.terminalFallingVelocity)
		{
			velocity.y = movement.terminalFallingVelocity;
		}
	}

	velocity.x = adjustedVelocity.x;

	// If we were jumping and our velocity became negative, set jumping flag to false:
	isJumping &= (velocity.y > 0);

	if (controller.BecameGroundedThisFrame)
	{
		HandleBecameGrounded();
	}
	else if (controller.LeftGroundThisFrame)
	{
		OnLeftGround?.Invoke();
	}
}
